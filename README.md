# Artfolio - Frontend

Frontend do projeto **Artfolio**, uma plataforma web voltada para divulgação de artes digitais, portfólios artísticos, planos pagos e interação entre usuários.

## Tecnologias utilizadas

- React
- Vite
- TailwindCSS 3
- FontAwesome
- JavaScript

---

## Pré-requisitos

Antes de iniciar, é necessário ter instalado:

- Node.js
- npm
- Git

Para verificar se estão instalados:

```bash
node -v
npm -v
git --version
